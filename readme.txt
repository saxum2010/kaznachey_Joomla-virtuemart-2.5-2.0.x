Требования:
1. Данный модуль интеграции с kaznachey разработан для Joomla версии 2.5;
2. Модуль разработан для компонента VirtueMart версии 2.0, т.е. перед установкой модуля установите этот компонент.

Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "ПОДКЛЮЧЕНИЕ"). 
После подписания договора быдут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля

Порядок установки модуля:
1. Данный модуль устанавливается в виде плагина. Для этого откройте "меню Расширения / Менеджер расширений";
2. В группе "Загрузить файл пакета" выберите файл данного архива и нажмите "Загрузить и установить";
3. В Менеджере плагинов включите плагин "Kaznachey Payment"
4. Откройте управление компонентом VirtueMart (меню Компоненты / VirtueMart) и нажмите "Способы оплаты";
5. Нажмите "Создать" и в открывшейся форме заполните поля:
 - Название платежа: Название способа оплаты, которое будут видеть ваши клиенты (Например: kaznachey);
 - Опубликовано: выберите "Да", если хотите, чтобы этот способ оплаты отобразился на сайте;
 - Описание платежа: Описание способа оплаты которое будет отображаться на сайте возле названия способа оплаты.
 - Способ оплаты: Выберите "Kaznachey Payment";
6. Сохраните и перейдите на вкладку "Настройки";
7. Заполните поля:
 - Merchant ID: Идентификатор вашего магазина в системе kaznachey, значение указано в панели управления магазином на сайте kaznachey.ua;
	- Идентификатор мерчанта - используется для подписи сообщений при переадресации (необходимо получить у менеджера после подписания договора);	
	- Секретный ключ мерчанта - используется для подписи скрытых сообщений на Result URL (необходимо получить у менеджера после подписания договора);	
8. Сохраните изменения.
